# Product Overview

Ask Author/Ask Channel is an AI-powered content discovery platform that enables users to query multimedia content (articles, videos, podcasts) through natural language conversations in Hindi, English, or transliteration.

## Core Value Proposition

Transform passive content archives into interactive knowledge bases where users can ask questions and receive synthesized answers with source citations, replacing traditional keyword search with semantic understanding.

## Key Features

- **Multilingual Semantic Search**: Query content in Hindi/English/transliteration with vector similarity matching
- **RAG-Powered Q&A**: LLM-synthesized answers grounded in actual content with mandatory source citations
- **Behavior-Based Personalization**: Content recommendations based on user history and preferences
- **WhatsApp-Style Chat UX**: Real-time streaming responses with voice input/output support
- **Responsible AI**: Grounded generation, source attribution, AI-synthesized labeling per India regulations

## Success Metrics

- Query success rate >90%
- Session length increase 3x vs traditional search
- Content reuse rate >40% from archives
- Query latency <3s (95th percentile)
- Scale: 10K concurrent users, 100K content assets
